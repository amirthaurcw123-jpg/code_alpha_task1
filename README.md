# 🌸 Iris Flower Classification using Machine Learning

## 📌 Project Overview

This project uses the Iris dataset to classify iris flowers into three different species using a Decision Tree Classifier.

## 🚀 Features

- Data Loading
- Data Preprocessing
- Train-Test Split
- Decision Tree Classification
- Model Evaluation
- Accuracy Score
- Confusion Matrix
- Classification Report

## 🛠️ Technologies Used

- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

1. Clone the repository.
2. Install the required libraries.

```bash

```

3. Open the notebook and run all cells.
